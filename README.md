## What is this?

A single-file slash command for [Claude Code](https://code.claude.com). Type `/red-coffee` and get:

- **Flow science** — how nitrate, caffeine, and your brain actually work
- **Recipes** — beetroot + arabica coffee blends calibrated for focus
- **Break nudges** — science-grounded reminders to step away and come back sharper

No install wizard. No dependencies. No account. One file.

## Install

**Option A — one command:**

```bash
mkdir -p ~/.claude/commands && curl -sL -o ~/.claude/commands/red-coffee.md https://raw.githubusercontent.com/akunkafoods/red-coffee-claude/main/red-coffee.md
```

**Option B — skills format (recommended for Claude Code v2+):**

```bash
mkdir -p ~/.claude/skills/red-coffee && curl -sL -o ~/.claude/skills/red-coffee/SKILL.md https://raw.githubusercontent.com/akunkafoods/red-coffee-claude/main/skills-format/red-coffee/SKILL.md
```

**Option C — manual:**

1. Download `red-coffee.md` from this repo
2. Drop it in `~/.claude/commands/` (or as `SKILL.md` in `~/.claude/skills/red-coffee/`)
3. Done

## Use

Open Claude Code. Type:

```
/red-coffee
```

That's it.

## What's Red Coffee?

Red Coffee is beetroot powder + Arabica coffee. Not a supplement. Not a nootropic stack. Just two real ingredients with independent, clinically-studied mechanisms:

**Beetroot → Nitrate → Nitric Oxide**
Dietary nitrate converts to NO via the enterosalivary pathway, dilating blood vessels and increasing blood flow — including to your brain.

**Coffee → Caffeine → Adenosine Blockade**
Caffeine occupies adenosine receptors, blocking the signal that makes you drowsy. It doesn't add energy — it removes the brake.

Each serving of Red Coffee delivers both pathways in one cup. No jitters. No crash. Just sustained, quiet focus.

Per serving (7g): ~58mg caffeine · ~116mg nitrate
